# AJFAN PIZZA

This repository contains the source for the AJFAN PIZZA front-end (Vite + React + TypeScript + Tailwind).

## Getting started

Prerequisites: Node.js (LTS) and npm.

```powershell
# from the repo root
cd .\ajfan-bites-translator-main
npm install
npm run dev
```

## Project structure

- `src/` - application source code (components, pages, styles)
- `public/` - static assets (images, `menu.pdf`, favicon)

## Scripts

- `npm run dev` - start dev server
- `npm run build` - build production assets
- `npm run preview` - preview production build

If you run `npm run dev` from the repository root, the top-level `package.json` proxies to the nested project folder.

## Deployment

Build the project and deploy the `dist/` contents to your static host (Netlify, Vercel, or similar).
